# Cosmitto Coffee Upgrade Ideas

Optional improvements the client can choose after the first delivery.

## Performance Polish

- Convert large PNG images to WebP or AVIF for faster loading.
- Compress hero videos further for smoother mobile playback.
- Add responsive image sizes for mobile and desktop.
- Add a lightweight loading state for slow networks.

## Content Upgrades

- Move menu items, prices, locations, and social links into a simple editable config file.
- Add seasonal menu sections for Ramadan, summer drinks, or limited offers.
- Add a featured product section that can be changed monthly.
- Add a printable or downloadable PDF menu.

## UX Polish

- Add per-route page titles for homepage, menu, and 404.
- Add a more polished hero video loading transition.
- Improve menu category navigation with stronger active-section feedback.
- Add subtle page transition animations between landing, menu, and 404.

## Business Features

- Add embedded Google Maps for the main location.
- Add QR menu support for tables.
- Add customer review highlights from Google or Facebook.
- Add newsletter or promotion signup if the client adds a backend service.

## Premium Upgrade Options

- Add a small CMS so the client can edit menu items without code.
- Add analytics with Vercel Analytics or Google Analytics.
- Add an admin-friendly `site.config.ts` file for brand, contact, links, and SEO.
- Add multi-language support for English, French, and Arabic.
- Add structured local-business pages for each Cosmitto location.

## Recommended Next Pick

The highest-value next improvement is an editable config file for menu, locations, phone, social links, and SEO. It makes future maintenance easier and makes the project feel more professional for handoff.
